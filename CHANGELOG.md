# Changelog

## v1.0.0 - Initial release

Initial public release of Aether, a minimal terminal radio player.

### Highlights

- Terminal UI built with Bubble Tea and Lip Gloss.
- Audio playback through mpv IPC.
- Default station set: BadRadio, Lofi Radio, Radio Paradise Main Mix FLAC.
- Radio Browser station search and preview flow.
- Station favorites and favorite tracks.
- Listening history with configurable retention.
- Stream state indicator: ON AIR, TUNING, BUFFERING, NO AUDIO, PAUSED.
- Desktop notifications for track changes.
- Clipboard integration with wl-copy/xclip/xsel fallbacks.
- Diagnostics via `aether doctor`.
- XDG config/data paths.
